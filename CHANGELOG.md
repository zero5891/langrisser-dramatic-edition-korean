# 변경 이력

## v0.13.17

### 랑그릿사 I UI

- 타이틀 메뉴 `START / LOAD / BACKUP RAM`을 `x=17` 공통 시작점에 좌측 정렬하고 글자 간격을 1픽셀로 통일
- 로드 화면 상단의 깨진 제목을 `LOAD`로 복구
- 배치·저장·고용·명령 화면의 `출격`, `결정`, `저장`, `공격` 깨짐과 오역 수정
- 용병 선택·결정 창의 용병명을 16×16 Neo둥근모로 확대하고 전용 캐시 페이지 24~27로 충돌 방지
- 전투 화면 직업명과 큰 `마법`을 실제 FONT/IMG 소비 경로에서 교정
- 지휘 메뉴 `공격 / 방어 / 수동`의 큰 글꼴을 Neo둥근모로 통일하고 작은 글꼴은 갈무리7 경로 유지
- `ENEMY PHASE` 명령열과 11개 표시 셀을 게임 원본과 바이트 단위로 동일하게 복구

### 빌드·검증

- 모든 변경을 불변 `v0.13.13` 베이스의 정확한 payload preimage에서 재설계
- 변경 payload를 `LANG1.BIN`, `LANG1/FONT.DAT`, `LANG1/IMG.DAT`, `LANG2/IMG.DAT`로 제한하고 v0.13.13 대비 ISO payload 변경을 정확히 `23개` 섹터로 고정
- 관련 자동 회귀검사 `27개`, payload 재추출, 멱등성, MODE1 framing, 서브채널, Track 2/XA/CDDA 및 6트랙 CUE 보존 검사 통과
- 정품 원본→v0.13.17 LDP 범위를 메인 채널 `2,991개` 섹터, `70,613개` 레코드, 교체 데이터 `6,673,154`바이트로 고정
- RetroArch 1.22.2의 Beetle Saturn 코어에서 용병명, 큰 `마법`, 원본 `ENEMY PHASE`, 지휘 메뉴 글꼴을 사용자 실기 확인
- v0.13.17부터 공식 화면 검수는 RetroArch의 Beetle Saturn 코어에서만 수행하며 Ymir를 포함한 다른 에뮬레이터와 실기는 공식 검증 대상에서 제외

## v0.13.13

### 랑그릿사 I

- Beetle Saturn 코어에서 시작 후 메인 메뉴가 표시되지 않던 문제 수정
- 실행 중 덮어써지는 제어 영역의 이전 HUD 훅을 제거하고 원본 복사 루틴 `0x0604A26C` 복구
- 하단 HUD 초기화기를 안전한 코드 영역 `0x06072210..0x06072313`으로 옮기고 새 부팅 전투 화면에서 안정성 게이트 통과
- 상태창의 `지휘범위 / 지휘보정` 구성 글리프를 정적 표시로 전환
- 배치 메인 행 4개와 제목 막대 3개, 총 7개 레이아웃을 정적 테이블로 전환
- 전투 명령 원형 메뉴의 `공격`을 원래 렌더러에서 정확히 표시하도록 교정
- 고정 UI 64셀의 본문 팔레트를 네이티브 흰색 인덱스로 교정하고 명령 메뉴 및 `아이템 장비 → 구입` 화면에서 흰색 표시 확인
- 알려진 `아이템 장비 → 구입` 경로는 캡처했으나, 호출 근거가 없는 과거 합성 `상점` 가드용 런타임 훅은 추가하지 않음

### 랑그릿사 II

- 시나리오용 16×16 글꼴 셀 `1,093개`를 KRPoc 기반 글꼴로 갱신
- 하단 HUD의 지연 초기화기를 안전한 코드 영역으로 이동

### 랑그릿사 I·II

- 시작 메뉴를 `START / LOAD / BACKUP RAM`으로 교체하고 제목 그림과 패널 장식 보존
- v0.13.12의 번역·Neo둥근모/갈무리 글꼴·음성·CDDA 계보 보존

### 빌드·검증

- v0.13.12 통합 이미지 대비 변경 ISO payload를 5개 파일, `76개` 섹터로 제한
- 최종 원본→v0.13.13 패치 범위를 메인 채널 `2,985개` 섹터, LDP `70,470개` 레코드로 고정
- 관련 자동 회귀검사 `55개` 통과
- 수동 LDP 적용기는 패치 전체 해시·manifest·레코드 범위·후행 데이터를 검증하고 임시 MDF를 검증한 뒤에만 최종 파일명으로 게시하도록 강화
- 수동 BIN/CUE 변환기도 임시 파일의 크기와 해시를 먼저 검증한 뒤 완성 파일명으로 게시하도록 강화
- RetroArch 1.22.2 / Beetle Saturn 1.32.1의 새 부팅 랑그릿사 I 전투 화면에서 하단 HUD `레딘 / 파이터` 표시와 런타임 바이트 감사 통과
- 모든 상점 분기, 게임 전체 완주, Ymir·다른 에뮬레이터·실기 최종 검증을 완료했다고 주장하지 않음

## v0.13.11

### 패치 도구

- 원본 오류, 공간 부족, 권한 문제와 기존 출력 파일을 쉬운 한국어로 안내
