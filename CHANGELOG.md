# 변경 이력

## v0.13.11

### 패치 도구

- Python과 명령어가 필요 없는 Windows 원클릭 GUI 패처 추가
- 원본 MDF 선택부터 해시 검사, 한글패치, 음성 호환 BIN/CUE 생성까지 한 번에 처리
- 패치 데이터를 EXE에 내장하여 일반 사용자는 실행 파일 하나만 사용
- 중간 패치 MDF를 만들지 않는 스트리밍 변환으로 필요 공간과 파일 혼란 축소
- 결과는 Ymir에서 바로 열 수 있는 권장 MODE2/XA CUE 하나만 생성
- 원본 오류, 공간 부족, 권한 문제와 기존 출력 파일을 쉬운 한국어로 안내

### 랑그릿사 I

- 저장·불러오기 화면과 첫 루시리스 대사의 깨짐 보완
- 고용창 용병 이름과 전투 화면 직업명 표시 보완
- 전투 명령의 `공격`과 상태창의 `지휘범위 / 지휘보정` 수정
- 마법명 `선더`를 `썬더`로 통일
- `SCENARIO / TURN` 영문 표기를 유지하면서 `TURN`의 U 글리프만 교정

### 랑그릿사 II

- 배치·시스템·설정·전투 명령의 고정 한글 57셀을 Neo둥근모로 통일
- 설정 화면의 잘못된 `난도상` 값을 `고속`으로 수정
- 동적 Neo둥근모 글꼴, 갈무리7 하단 HUD 글꼴과 상점 오류 수정 계보 보존

### 디스크·음성

- v0.12.25 이후 번역·음성·능력치·전투 연출 수정 계보 유지
- MODE2/XA 이벤트 음성, CDDA 배경음 및 MDF 서브채널 보존

## v0.13.7

### 랑그릿사 I

- 본편 대사와 주요 메뉴·전투 UI 한국어화
- 단어 사이 공백을 8픽셀 반각 기준으로 정리
- 큰 한글 글꼴을 Neo둥근모 16px 기반 비트맵으로 교체
- 작은 한글 글꼴을 갈무리7 8px 기반 비트맵으로 교체
- 루시리스 문답의 갑작스러운 평서체를 존댓말로 통일
- 구매·판매, 시스템, 설정, 명령, 배치 화면의 미번역·깨짐 수정
- `SCENARIO`, `TURN`, `PCM`, `ON`, `OFF` 영문 표기 유지
- 전투 하단 HUD의 이름·직업명 및 고용창 표시 보완

### 랑그릿사 II

- 큰 한글 글꼴을 Neo둥근모 16px 기반 비트맵으로 교체
- 하단 HUD용 작은 한글 글꼴을 갈무리7 8px 기반 비트맵으로 교체
- 장착품만 남은 상태에서 판매창 진입 시 발생하던 SH-2 오류 수정
- 빈 판매 목록 안내를 `판매품없음`으로 단축
- 구매 시 소지품이 가득 찬 안내를 `소지품가득`으로 단축

### 디스크·음성

- v0.12.25의 번역·메뉴·직업명·음성 복원·능력치·`상쇄→마법` 수정 계보 유지
- MODE2/XA 이벤트 음성, CDDA 배경음 및 MDF 서브채널 보존

## v0.12.25

- 마법 사용 전투 연출의 오역 `상쇄`를 `마법`으로 수정
- IMG.DAT의 해당 32×16 고정 그래픽만 교체

## v0.12.24

- 캐릭터 전용 강화 클래스 수치는 보존

## v0.12.23

- 후반부 대사·내레이션의 글자 깨짐, 밀림, 잘림 및 제어코드 노출 수정
- 대화창 14칸·페이지 3줄 기준 적용
- `폐`, `게`, `탱`, `기`, `배` 글리프 교체

- LOAD 화면 접미 문구를 `계속`으로 변경
